# Ссылки на файлы в Yandex Disk

Этот файл содержит ссылки на тяжелые файлы, хранящиеся в Yandex Disk.

## Инструкция по получению публичной ссылки

1. Откройте Yandex Disk
2. Найдите нужный файл
3. Правой кнопкой мыши → "Получить ссылку" или "Поделиться"
4. Скопируйте ссылку и вставьте в соответствующий раздел ниже

## Примечания

- Убедитесь, что ссылки имеют публичный доступ (или доступ по ссылке)
- Для больших файлов рекомендуется использовать прямые ссылки на скачивание
- Если файл очень большой, можно разбить его на части и добавить ссылки на каждую часть
